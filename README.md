### Installation
```

pip install tensorflow

pip install yattag

pip install torch_geometric
pip install nltk spacy==2.3.7
python3 -m spacy download en_core_web_md
pip install benepar
```

source activate anntf2
python3 SANIHFNLPpy_main.py
